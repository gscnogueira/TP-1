Descrição detalhada

Padrão para representação de Número de conta bancária.

Requezitos de Formato

Para ser considerado válido, a cadeia de caracteres a ser passada deve ter o formato XXXXXX-Y, onde X é digito e Y é digito verificador. O método utilizado para calculo do digito de verificção é o Módulo 10.
